# Today Safety Docs

공공데이터 기반 AI 작업 전 안전위험 분석 및 체크리스트 생성 서비스 문서 저장소입니다.

## 개요

본 서비스는 산업현장의 작업 조건을 입력하면 공공데이터 기반 AI가 주요 사고유형과 위험도를 분석하고, 그 결과를 바탕으로 작업 전 안전 체크리스트와 브리핑 문안을 자동 생성하는 것을 목표로 합니다.

이 아이디어는 단순한 조건 분기형 점검표 생성기가 아니라, 공공데이터 기반 위험분석 결과를 현장에서 바로 활용 가능한 실행 문서로 전환하는 데 초점을 둡니다.

## 핵심 아이디어

- 작업 조건 기반 위험도 분석
- 주요 사고유형 예측 및 우선순위 도출
- 유사 재해사례 기반 설명 제공
- 작업 전 안전 체크리스트 자동 생성
- 현장 반장 및 관리자용 브리핑 문안 생성

## 구현 방향 요약

- 프론트엔드: Next.js + TypeScript
- 백엔드: FastAPI + Pydantic
- AI 계층: Python 백엔드 내부 서비스 모듈
- 데이터 저장소: PostgreSQL
- 외부 AI 연동: OpenAI Responses API

초기 단계에서는 프론트엔드와 백엔드를 분리하고, AI는 별도 독립 서비스로 떼지 않고 백엔드 내부 계층으로 포함하는 구조를 기본 전략으로 한다.

## 목표

형식적인 안전점검을 실질적인 사전 예방 활동으로 전환하고, 중소사업장을 포함한 다양한 산업현장에서 작업 전 위험 인지와 현장 브리핑 품질을 높이는 것이 목표이다.
