# Priority Review Guide

이 문서는 `taxonomy_review_workbook.xlsx`를 검수할 때 시간을 줄이기 위한 빠른 판단 가이드입니다.

핵심 원칙은 두 가지입니다.

- 사고유형은 `어떻게 다쳤는가`를 본다.
- 공정은 `무슨 작업 중이었는가`를 본다.

직종명만 있고 작업 장면이 없으면 공정은 무리해서 넣지 말고 `기타` 또는 `skip`으로 두는 편이 낫습니다.

## 가장 먼저 할 일

1. `Accident Review`에서 상위 100개를 먼저 본다.
2. 그다음 `Process Review`에서 상위 100개를 본다.
3. 아래의 “빠른 처리 규칙”에 맞는 것은 빠르게 정리한다.
4. 애매한 것만 `sample_*` 컬럼을 자세히 읽는다.

## 빠른 처리 규칙

### 사고유형 빠른 규칙

- 아래 키워드는 대부분 `충돌`로 보면 됩니다.
  - `골절`
  - `열린상처`
  - `표재성 손상`
  - `두개내손상`
  - `경막하출혈`
  - `지주막하출혈`
  - `경막외출혈`
  - `탈구`
  - `치아 파절`
- 아래 키워드는 대부분 `무리한동작`으로 보면 됩니다.
  - `인대의 파열`
  - `힘줄의 손상`
  - `추간판전위`
  - `아킬레스힘줄`
  - `방아쇠 손가락`
  - `유착성 관절낭염`
- 아래 키워드는 현 taxonomy로는 보통 `기타`가 맞습니다.
  - `청력소실`
  - `만성 폐색성 폐질환`
  - `악성 신생물`
  - `적응장애`
  - `뇌경색증`
  - 원문이 너무 포괄적인 `질병`
- 아래 표현은 문맥을 꼭 같이 봐야 합니다.
  - `사고`
  - `깔림`
  - 안전장치/전원차단/점검 관련 문장형 표현

### 공정 빠른 규칙

- 아래처럼 “직종명만” 있는 경우는 대체로 `기타` 또는 `skip`입니다.
  - `건설 및 광업 단순 종사원`
  - `기타 기능관련 종사자`
  - `기타 서비스관련 단순 종사원`
  - `행정 사무원`
  - `사회복지관련 종사자`
  - `보건의료관련 종사자`
  - `매장 판매 종사자`
  - `영업종사자`
- 아래처럼 작업 장면이 드러나는 것은 바로 분류 가능합니다.
  - `설비, 구조물 등 높은 장소에서의 작업` -> `설치/해체`
  - `위험물질 취급 또는 그 장소에서 작업` -> `화학물질취급`
  - `벌목·조재 작업` -> `절단/가공`
  - `지붕, 천장 등 위에서 작업` -> `설치/해체`
  - `건설기계 사용 작업` -> `굴착/토공` 또는 `운반/인양`

## 검수 전략 추천

### 1차 빠른 정리

- 사고유형:
  - 질병/직업병 계열은 먼저 `기타`
  - 열린상처/골절/두부외상 계열은 먼저 `충돌`
  - 인대/힘줄/디스크 계열은 먼저 `무리한동작`
- 공정:
  - 직종명만 있는 것은 `기타` 또는 `skip`
  - 작업 문장형 표현만 집중 검토

### 2차 정밀 검토

- `sample_narrative_text`가 있는 문장형 항목
- `깔림`, `사고`, 안전장치 관련 표현
- 물류/이동과 상하차가 섞이는 직종

## 최종 팁

- 사고유형은 너무 세부 상병명에 끌려가기보다 `다친 메커니즘`을 먼저 보세요.
- 공정은 직종명과 작업공정을 혼동하지 않는 것이 가장 중요합니다.
- 확신이 낮으면 `skip`으로 넘기고, 일관성 있게 처리하는 편이 더 좋습니다.
