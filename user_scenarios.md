# 사용자 시나리오

## 서비스 개요

본 서비스는 산업현장의 작업 조건을 입력하면, 공공데이터 기반 AI가 주요 사고유형과 위험도를 분석하고 작업 전 안전 체크리스트와 브리핑 문안을 생성하는 서비스이다.

사용자 입장에서는 복잡한 안전자료를 직접 해석하는 대신, 오늘 작업에 맞는 핵심 위험과 행동지침을 바로 받는 흐름으로 설계된다.

## 기본 사용자 흐름

1. 현장관리자 또는 반장이 서비스에 접속한다.
2. 오늘의 작업 정보를 입력한다.
3. 시스템이 공공데이터와 외부 환경데이터를 조회한다.
4. AI가 위험도와 사고유형을 분석한다.
5. 결과 화면에서 체크리스트와 브리핑 문안을 확인한다.
6. 현장에서 점검 및 브리핑 후 작업을 시작한다.

## 공통 입력 항목

- 산업군
- 직종
- 작업 장소
- 실내 또는 실외 여부
- 작업 공정 또는 작업 유형
- 날씨
- 고소작업 여부
- 화학물질 사용 여부
- 중장비 사용 여부

## 공통 출력 항목

- 오늘의 위험도: 낮음 / 보통 / 높음
- 주요 사고유형 Top 3
- 위험 상승 요인
- 작업 전 안전 체크리스트
- 필수 보호구
- 작업중지 또는 재검토 조건
- 유사 재해사례 요약
- 현장 브리핑 문안

## 시나리오 1. 건설현장 반장

### 사용자

- 건설현장 반장

### 상황

아침 작업 시작 전, 철골 작업과 자재 인양 작업이 예정되어 있다. 실외 작업이며 강풍 예보가 있다.

### 입력

- 산업군: 건설
- 직종: 철골 작업
- 작업 장소: 실외
- 작업 공정: 자재 인양
- 날씨: 강풍
- 고소작업 여부: 예
- 중장비 사용 여부: 크레인 사용

### 시스템 분석 결과

- 위험도: 높음
- 주요 사고유형: 낙하, 충돌, 추락
- 위험 상승 요인: 강풍, 중량물 인양, 실외 고소작업

### 생성 결과

- 인양구 결속 상태 확인
- 작업 반경 출입통제 여부 확인
- 고소작업자 안전대 착용 여부 확인
- 풍속 기준 초과 시 작업 중지 여부 검토
- 낙하물 방지조치 확인

### 브리핑 문안 예시

`오늘은 강풍으로 인해 자재 낙하와 충돌 위험이 큽니다. 인양 전 결속 상태와 작업 반경 통제를 우선 확인하고, 풍속이 기준을 넘으면 즉시 작업을 재검토하십시오.`

## 시나리오 2. 제조업 설비 관리자

### 사용자

- 제조업 생산라인 설비 관리자

### 상황

설비 정비 작업이 예정되어 있으며 절단기 점검과 윤활유 사용이 포함되어 있다.

### 입력

- 산업군: 제조
- 직종: 설비 정비
- 작업 장소: 실내
- 작업 공정: 절단기 점검
- 화학물질 사용 여부: 예
- 화기 사용 여부: 예

### 시스템 분석 결과

- 위험도: 보통~높음
- 주요 사고유형: 끼임, 화재, 화학물질 노출
- 위험 상승 요인: 잔류 에너지, 화기 사용, 윤활유 취급

### 생성 결과

- 설비 전원 차단 여부 확인
- 잔류 에너지 해제 여부 확인
- 방염장갑 및 보안경 착용 여부 확인
- 환기 상태 점검
- 소화기 비치 상태 확인

### 브리핑 문안 예시

`오늘 작업은 설비 끼임과 화재 위험이 있으므로 정비 전 전원 차단과 잔류 에너지 해제를 먼저 확인하고, 화기 사용 구간에서는 환기와 소화장비 상태를 반드시 점검하십시오.`

## 시나리오 3. 물류센터 팀장

### 사용자

- 물류센터 현장 팀장

### 상황

우천으로 인해 바닥이 젖은 상태에서 지게차를 이용한 상하차 작업이 진행된다.

### 입력

- 산업군: 물류
- 직종: 지게차 작업
- 작업 장소: 실내
- 작업 공정: 상하차
- 날씨: 우천
- 중장비 사용 여부: 예

### 시스템 분석 결과

- 위험도: 보통
- 주요 사고유형: 충돌, 전도, 미끄러짐
- 위험 상승 요인: 젖은 바닥, 지게차 이동 동선, 적재물 시야 제한

### 생성 결과

- 바닥 젖음 구간 표시 여부 확인
- 지게차 이동 동선과 보행 동선 분리 여부 확인
- 적재물 높이와 시야 확보 여부 확인
- 작업자 반사조끼 착용 여부 확인

### 브리핑 문안 예시

`우천 영향으로 바닥이 미끄러우므로 지게차 전도와 작업자 충돌 위험이 있습니다. 젖은 구간 표시, 동선 분리, 적재 시야 확보를 우선 확인한 뒤 작업을 진행하십시오.`

## 핵심 사용자 가치

- 오늘 작업에 맞는 위험요인을 빠르게 파악할 수 있다.
- 방대한 안전자료를 읽지 않고도 실행 가능한 점검 항목을 확인할 수 있다.
- 현장 브리핑의 품질과 일관성을 높일 수 있다.
- 중소사업장에서도 안전관리 준비 시간을 줄일 수 있다.

## 서비스의 본질

이 서비스는 사용자가 안전자료를 많이 읽도록 만드는 것이 아니라, 오늘 작업에 필요한 안전조치를 바로 실행할 수 있도록 돕는 데 목적이 있다.
